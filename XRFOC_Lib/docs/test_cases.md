# XRFOC Test Cases

## 1. 目标

建立 XRFOC 与 MCSDK v5.4.4(F0) 的可复现实验对标，覆盖启动、切闭环、稳态、动态、保护和恢复。

## 2. 采集信号（最低要求）

- 时间戳（us）
- `Ia/Ib/Ic`
- `I_alpha/I_beta`
- `I_d/I_q`
- `U_d/U_q`
- `Est_Theta`（无感）
- `Theta_sensor`（有感基准）
- 机械速度（rpm）
- `Vbus`
- 状态机阶段（V/F、ALIGN、CLOSE_LOOP）

## 3. 回归矩阵

### TC-01 启动成功率
- 条件：空载，目标 500 rpm。
- 指标：10 次上电启动成功次数。
- 通过标准：`>= 9/10`。

### TC-02 I/F 启动回退
- 条件：提高负载或降低母线使 I/F 难以闭环。
- 指标：是否触发 I/F -> V/F 回退并最终闭环。
- 通过标准：回退触发正确、系统不锁死。

### TC-03 角度误差
- 条件：有感对照，500/1000/1500 rpm。
- 指标：`|Est_Theta - Theta_sensor|` 均值与峰值。
- 通过标准：均值 `< 10°elec`，峰值 `< 25°elec`。

### TC-04 稳态速度纹波
- 条件：目标 1000 rpm，稳态 10 s。
- 指标：速度标准差与峰峰值。
- 通过标准：纹波不高于 MCSDK 对照的 1.3 倍。

### TC-05 动态阶跃
- 条件：500 -> 1500 rpm，再 1500 -> 500 rpm。
- 指标：超调、调节时间、是否丢步。
- 通过标准：不丢步；调节时间不高于 MCSDK 的 1.5 倍。

### TC-06 过流保护
- 条件：高负载突加。
- 指标：是否在保护阈值内触发，恢复后可重启。
- 通过标准：保护触发正确，重启可恢复。

### TC-07 高速上限
- 条件：逐步提速直至极限。
- 指标：可持续运行最高转速。
- 通过标准：达到目标平台要求，且温升/电流在安全范围。

## 4. 与 MCSDK(F0) 对标输出格式

每个测试用例输出：

- XRFOC 数值
- MCSDK 数值
- 比值（XRFOC/MCSDK）
- 结论（PASS/FAIL）
- 备注（参数、供电、电机型号、环境温度）

## 5. 参数记录模板

- 电机：型号、极对数、Rs、Ls、Ke
- 硬件：驱动板、采样电阻、ADC 增益、PWM 频率、死区
- XRFOC：PID、SMO/PLL 参数、启动模式参数
- MCSDK：同工况参数导出值
