## 2. 编码器接口选型与信号定义

### 2.1 编码器选型推荐

| 参数 | 磁编码器 (推荐) | 光编码器 | 霍尔传感器 |
|------|-----------------|---------|-----------|
| **型号** | AS5048A / MT6825 | E6B2-CWZ3E | SS49E x3 |
| **接口** | SPI (最高10MHz) | ABZ正交 | 数字IO x3 |
| **分辨率** | 14-bit (16384 CPR) | 1000~8192 PPR | 6步/电周期 |
| **精度** | ±0.05° | ±0.1° | ±30°电角度 |
| **适用场景** | 关节精确位置控制 | 中等精度 | 仅换相 |
| **成本** | ¥15~30 | ¥50~200 | ¥3 |

### 2.2 推荐方案: AS5048A (SPI磁编码器)

**信号定义表**:

| MCU Pin | 信号 | 方向 | 说明 |
|---------|------|------|------|
| PA5 (SPI1_SCK) | CLK | OUT | SPI时钟, ≤10MHz |
| PA6 (SPI1_MISO) | MISO/DATA | IN | 编码器角度数据 |
| PA7 (SPI1_MOSI) | MOSI | OUT | 命令字 |
| PB6 | CS_n | OUT | 片选, 低有效 |
| N/A | VDD | PWR | 3.3V或5V |
| N/A | GND | PWR | 共地 |

**数据帧格式**: 
```
TX: [0xFFFF] (读角度寄存器)
RX: [PARITY|ERROR|D13:D0]  → angle = (D13:D0) * 360° / 16384
```

### 2.3 当前无感方案 (SMO) 信号

当前系统使用滑模观测器(SMO)进行无传感器控制，编码器为可选项。如果安装编码器:
- 低速时使用编码器角度 (< 阈值转速)
- 高速时使用SMO估算角度 (> 阈值转速)
- 可实现 smooth handover

## 3. 三层控制环时序图

```
时间轴 →  ────────────────────────────────────────────────────────────────►
                                                                          
TIM1 Center-Aligned PWM (18kHz, period = 55.6μs)
          ▕▔▔▔╲    ╱▔▔▔▕▔▔▔╲    ╱▔▔▔▕▔▔▔╲    ╱▔▔▔▕
PWM_CH1   ▕    ╲  ╱    ▕    ╲  ╱    ▕    ╲  ╱    ▕
          ▕     ╲╱     ▕     ╲╱     ▕     ╲╱     ▕
          ├─────────────┼─────────────┼─────────────┤
          0          55.6μs       111.2μs       166.8μs

TIM1_CC4 Trigger (ADC Injection)
          ▕  ↑          ▕  ↑          ▕  ↑          ▕
          ▕  │          ▕  │          ▕  │          ▕
          ▕  ADC_start  ▕  ADC_start  ▕  ADC_start  ▕

ADC ISR (@ 18kHz = 每55.6μs执行一次):
  ┌────────────────────────────────────────────┐
  │ t=0μs:  读取 ADC Injected Rank 1/2/3      │
  │ t=1μs:  减偏置 + 低通滤波 (Ia, Ib, Ic)     │
  │ t=3μs:  Clarke变换 → Iα, Iβ               │
  │ t=4μs:  Park变换 → Id, Iq (用Est_θ)       │
  │ t=5μs:  SMO观测器更新 → Est_θ, Est_ω       │
  │ t=8μs:  ▼ 电流环 PI (Id→Ud, Iq→Uq)        │  ← 电流环 @ 18kHz
  │ t=10μs: 逆Park → Uα, Uβ                   │
  │ t=11μs: SVPWM → Ta, Tb, Tc → CCR1/2/3     │
  │ t=13μs: 状态机调度                          │
  └────────────────────────────────────────────┘
                                                    
速度环 (分频执行 @ 2~5kHz, 每 N 个 PWM 周期执行一次):
  ┌──────────────────────────────┐
  │ 速度误差 = ω_ref - Est_ω    │
  │ Iq_ref = Speed_PID(error)   │
  └──────────────────────────────┘
  执行时机: motor_I_UVW.cntt % speed_div == 0

位置环 (@ 500Hz, 主循环或独立定时器):
  ┌──────────────────────────────────────┐
  │ 位置误差 = θ_ref - θ_measured        │
  │ ω_ref = Pos_PID(error)              │
  │ + 前馈: ω_ff = dθ_ref/dt            │
  └──────────────────────────────────────┘
  执行时机: 2ms 定时器中断 (TIM6)
```

### 3.1 控制环参数一览

|--------|------|------|------|------|------|

## 4. 容错与安全策略

### 4.1 故障分类与响应

| 故障类别 | 检测方法 | 响应动作 | 恢复条件 |
|----------|----------|----------|----------|
| **过流** (>10A) | ADC看门狗 + 软件比较 | 立即SHUTDOWN + PWM=0 | 手动复位 |
| **过压** (>30V) | ADC CH0 (Vbus) > 阈值 | SHUTDOWN + 刹车电阻 | Vbus回落 |
| **欠压** (<18V) | ADC CH0 < 阈值 | 降功率运行 | Vbus恢复 |
| **过温** (>85°C) | ADC CH1 (NTC) > 阈值 | 降速运行→停机 | 温度回落 |
| **堵转** | 连续N周期|ω|<阈值且|Iq|>限 | 降力矩→停机 | 外力解除 |
| **观测器失锁** | |Est_θ_error| > 45° 持续 | 回退VF重启 | 自动恢复 |
| **编码器故障** | SPI通信超时/CRC错 | 切换SMO估算 | 通信恢复 |
| **通信丢失** | 上位机心跳超时>100ms | 减速→停机 | 心跳恢复 |

### 4.2 安全状态机

```
   ┌─────────┐     上电      ┌──────────┐
   │  INIT   │──────────────▶│ STANDBY  │
   └─────────┘               └─────┬────┘
                                   │ enable cmd
                                   ▼
                             ┌──────────┐
                     ┌───────│ VF_START │◄──────────┐
                     │       └─────┬────┘           │
                     │             │ VF完成          │ fallback
                     │             ▼                 │
                     │       ┌──────────┐           │
                     │       │  ALIGN   │───────────┘
                     │       └─────┬────┘
                     │             │ 对齐OK
                     │             ▼
                     │       ┌──────────┐
                     │       │ RUNNING  │
                     │       └─────┬────┘
                     │             │ fault
                     │             ▼
                     │       ┌──────────┐
                     └──────▶│  FAULT   │
                             └─────┬────┘
                                   │ clear + reset
                                   ▼
                             ┌──────────┐
                             │ STANDBY  │
                             └──────────┘
```

### 4.3 关键保护实现要点

1. **硬件保护优先**: TIM1 Break输入 (BKI) 连接比较器输出，实现 <100ns 硬件级过流保护
2. **双冗余**: 软件过流检测作为第二道防线
3. **安全停机序列**: 
   ```
   SHUTDOWN_OFF → CCR=0 → MOE_DISABLE → run_flag=STOP → 设置FAULT标志
   ```
4. **禁止动态内存**: 所有故障状态使用静态标志位，无 malloc/new
